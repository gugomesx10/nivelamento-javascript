# Nivelamento JavaScript

Repositório com os exercícios de nivelamento em JavaScript.

Esse repositório contém as atividades práticas de JavaScript organizadas por tema:

- Console.log básico
- Operações matemáticas

### EX02 - Variáveis e tipos
- Uso de `const` e `let`
- Operador `typeof`

### EX03 - Valores especiais
- Trabalhando com `null` e `undefined`
- Propriedades de objetos

### EX04 - Arrays e objetos
- Manipulação de arrays
- Criação e acesso a objetos

### EX05 - Estruturas condicionais
- `if/else if/else`
- `switch/case`

### EX06 - Loops
- Loop `for`
- Loop `while`

### EX07 - Funções
- Funções tradicionais e arrow functions
- Escopo, callbacks e funções assíncronas
